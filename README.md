# Tempo Support & Developer Issues

This repository is the central place for developers and users to open issues, ask questions, and request guidance across the Tempo ecosystem.

If you are unsure which Tempo repository your issue belongs to, open it here and we will triage it to the appropriate project. Make sure to checkout our [docs](https://docs.tempo.xyz) before filing an issue. 

## What you can do here
- Report possible bugs
- Ask development or integration questions 
- Request clarifications on documentation or APIs
- Suggest improvements

## What this repository is not
- It is not the implementation repository for Tempo Chain or Tooling
- It is **not** a security disclosure channel

## Security
If you believe you've found a security vulnerability, please do not report it via GitHub issues. Instead, email security@tempo.xyz and we will acknowledge your report within 5 days and provide a more detailed follow-up within 10 days.

## Troubleshooting

- Fee token: simulate OK but send FAIL? See docs/fee-token-troubleshooting.md
